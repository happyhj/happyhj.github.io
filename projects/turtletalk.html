TurtleTalk: Voice-Driven Programming Game

NAVER Clova AI · ACM CHI 2019

Overview

TurtleTalk is a voice-driven educational programming game that teaches children foundational programming concepts — sequencing and iteration — through spoken commands. The child speaks instructions like "go forward" or "turn left" to control a turtle on a visual canvas, seeing their program execute in real time.

I designed and built the full-stack prototype. The work was published at ACM CHI 2019, one of the premier venues in Human-Computer Interaction research.

Problem

Young children who haven't yet learned to type still benefit from early exposure to computational thinking — understanding sequences of steps, repetition, and cause-and-effect relationships in code. Existing programming tools for kids (Scratch, Code.org) require reading, clicking, or dragging blocks, which creates barriers for pre-literate or early-literate children.

Solution

A browser-based prototype combining three web APIs into a seamless voice-to-code loop:

  • Web Speech API for real-time voice command recognition — the child speaks naturally, and the system interprets directional and control commands.
  • Google Translate API for multilingual support — allowing children to speak commands in their native language (Korean, English, etc.) with automatic translation to the command set.
  • Canvas-based turtle graphics for immediate visual feedback — the turtle moves, draws, and turns on screen as commands are spoken, providing a direct link between instruction and outcome.

The entire prototype runs in the browser with no installation required, making it accessible on any device with a microphone.

My Role

  • Designed the interaction model: voice → parsed command → visual execution
  • Built the full-stack prototype (frontend + voice processing pipeline)
  • Implemented the multilingual command mapping via translation API
  • Designed the visual turtle graphics rendering and animation
  • Participated in user study design and prototype iteration

Impact

  • Published at ACM CHI 2019 (Glasgow, UK) — the top venue in Human-Computer Interaction
  • Demonstrated that voice-based programming interfaces are viable for pre-literate children
  • Prototype validated through user studies with children
